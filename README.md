# firstgame.js

Recreation of my first game and first project. I took some parts from my (now privated) ECS game engine.

> Though, I plan to remake that game engine and add some performance enhancements (AABB trees, maybe?).

## Controls

| Key | Function     |
| :-- | :----------- |
| W   | Forward      |
| A   | Rotate Left  |
| S   | Backward     |
| D   | Rotate Right |
| LMB | Shoot        |
